🏨 Select Hotel Product Widget - 개발 환경

📋 기본 사용 예제

실제 Sabre API를 호출하는 호텔 상품 위젯입니다.

<select-hotel-product sabre-id="383336" check-in="2025-08-15" nights="1" num-of-people="2" />

📋 usage


required attributes

  • sabre-id - Sabre 시스템의 호텔 ID (숫자)
  • check-in - 체크인 날짜 (YYYY-MM-DD 형식)
  • api-base-url - API 서버 주소

optional attributes

  • nights - 숙박 일수 (기본값: 1)
  • num-of-people - 인원 수 (기본값: "2")

📋 npm script commands

  • build:widget - react component로 부터 web component 패키지를 번들링 합니다.
  • dev:widget - react component의 수정사항을 HMR로 web component 빌드하는 로컬 개발 서버를 구동시킵니다.
