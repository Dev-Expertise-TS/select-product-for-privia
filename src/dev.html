🏨 Select Hotel Product Widget - 개발 환경

기본 사용 예제

실제 Sabre API를 호출하는 호텔 상품 위젯입니다.

<select-hotel-product sabre-id="12345" check-in="2024-03-15" nights="2" num-of-people="2" api-base-url="https://api.example.com"> </select-hotel-product>

커스텀 제목과 혜택

제목과 혜택을 커스터마이징한 예제입니다.

<select-hotel-product sabre-id="67890" check-in="2024-04-01" nights="3" num-of-people="4" api-base-url="https://api.example.com" prd-title="프리미엄 패키지 - 특별 혜택" benefits='[{"icon":"🍽️","benefit":"4인 조식 포함"},{"icon":"💳","benefit":"$200 크레딧 제공"}]' cautions='["환율 변동에 따라 최종 금액이 달라질 수 있습니다."]'> </select-hotel-product>

테스트용 (목업 API)

개발 중 테스트를 위한 목업 API 사용 예제입니다.

📋 사용법

필수 속성들:

  • sabre-id: Sabre 시스템의 호텔 ID (숫자)
  • check-in: 체크인 날짜 (YYYY-MM-DD 형식)
  • api-base-url: API 서버 주소

선택적 속성들:

  • nights: 숙박 일수 (기본값: 1)
  • num-of-people: 인원 수 (기본값: "2")
  • prd-title: 상품 제목
  • benefits: 혜택 목록 (JSON 문자열)
  • cautions: 주의사항 목록 (JSON 문자열)
