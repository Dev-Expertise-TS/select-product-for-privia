# 럭셔리 셀렉트 호텔 객실 상품 컴포넌트

이 프로젝트는 럭셔리 셀렉트 호텔 객실 상품을 표시하는 컴포넌트를 제공합니다.

## 컴포넌트 목록

### React 컴포넌트
- **SelectHotelProductItem**: Sabre API를 통해 실시간 가격 정보를 가져와 표시하는 React 컴포넌트

### 웹 컴포넌트  
- **select-hotel-product**: React 컴포넌트를 웹 컴포넌트로 변환한 버전으로, 프레임워크에 구애받지 않고 사용 가능

## 주요 기능

- 실시간 호텔 가격 조회
- 반응형 디자인
- 카카오톡 상담 연동
- 취소 정책 표시
- 로딩 상태 처리
